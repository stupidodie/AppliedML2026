PyTorch NN Regression v7 (ResMLP + Weighted Ensemble) - Guanran Tai
======================================================================

Objective: Electron energy regression (log-transformed, combined loss)
Metric: Relative MAE = mean(|E_pred - E_true| / E_true)

Architecture:
  Input: 20 features
  Hidden: 512 -> 5xResBlock(SiLU+BatchNorm+Dropout) -> 256 -> 128 -> 1
  Activation: SiLU + BatchNorm + Dropout(0.08)
  Init: Kaiming normal
  Parameters per model: ~1,300,000

Loss: Huber(log-space, delta=0.3) + 2.0 x RelativeMAE(direct)
Sampler: WeightedRandomSampler(1/E), replacement=True
Optim: AdamW(lr=3e-4, wd=1e-3) + 15-epoch warmup
Scheduler: CosineAnnealingWarmRestarts(T_0=25xsteps, T_mult=2)
Ensemble: 14 models, weighted averaging (softmax over -rel_mae)
Device: MPS (Apple M4)
Training data: 37,718 electrons (E > 1000 MeV)
