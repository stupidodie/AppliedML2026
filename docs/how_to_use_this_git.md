# Working with the code repository

### How to get files:
If you simply want to access files in a GitHub repository, then you start by "connecting" to this with the command: "git clone https://github.com/AppliedMachineLearningNBI/AppliedML2026.git ." (see installation instructions, e.g. point 3 in [install_instructions_vscode.md](install_instructions_vscode.md)).

Once this is setup you may anywhere in this directory and all subdirectories underneath at any time write "git pull", and you will get new files and the latest versions. Note that if you changed something, it will warn you, before overwriting. If you want to overwrite, just remove your (changed) file. As you surely want to develop code, renaming it is the way forward.
